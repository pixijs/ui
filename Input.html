Class: Input

Input

Container based component that creates an input element, so we can read users input text.

Example

```
new Input({
    bg: new PixiSprite(Texture.from('item.png')),
    padding,
    textStyle: {
        ...buttonTextStyle,
        fill: textColor,
        fontSize,
    },
    maxLength,
    align,
    placeholder,
    value: text,
});
```
